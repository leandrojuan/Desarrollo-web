Sobre Mí

Hola, soy Leandro Juan. Como estudiante de ingeniería en computación con interés en el desarrollo web, estás enfocado en aprender y desarrollar habilidades en este campo. Los roles disponibles para programadores juniors incluyen el diseño de sitios web, desarrollo de páginas web y creación de aplicaciones web.

Las habilidades esenciales incluyen un conocimiento profundo de lenguajes de marcado y programación como HTML, CSS y JavaScript, que son la base para estructurar, diseñar y dotar de interactividad a las páginas web. Además, es fundamental tener experiencia con frameworks y librerías populares como React, Angular o Bootstrap, que aceleran el desarrollo y mejoran la calidad del código.
